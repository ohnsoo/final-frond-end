# UnklabMarketPlace

Project ini adalah marketplace untuk Universitas Klabat.

## Teknologi yang Digunakan

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Cara Menjalankan Project

1.  Clone repository ini.
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Jalankan development server:
    ```sh
    npm run dev
    ```

## Konfigurasi

Pastikan Anda memiliki file `.env` dengan konfigurasi Supabase yang benar:

```env
VITE_SUPABASE_PROJECT_ID="project-id-anda"
VITE_SUPABASE_PUBLISHABLE_KEY="anon-key-anda"
VITE_SUPABASE_URL="https://project-id-anda.supabase.co"
```
